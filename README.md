The easy 1 stop shop for installation.

# Parsec installation guide for Windows

 Step 1: Visit the official Parsec website
 Open your browser and go to: https://parsec.app/downloads

 Step 2: Download the Windows installer
 Click the "Download for Windows" button to get the installer (.exe file).

 Step 3: Run the installer
 Locate the downloaded file (usually in your Downloads folder) and double-click it.

 Step 4: Follow the installation prompts
 Accept the license agreement and choose your installation preferences.

 Step 5: Launch Parsec
 Once installed, open Parsec from the Start Menu or desktop shortcut.

 Step 6: Sign in or create an account
 You’ll need a Parsec account to use the app. Sign in or register if you don’t have one.

 Step 7: Configure settings (optional)
 Adjust resolution, bandwidth, and controller settings as needed in the app’s settings menu.


 # Parsec Installation Guide for Linux

 Step 1: Download the Parsec .deb package
 Visit https://parsec.app/downloads and download the Linux (.deb) installer.

 Step 2: Open a terminal and navigate to the download location
```cd ~/Downloads```

#Step 3: Install the package using dpkg
```sudo dpkg -i parsec-linux.deb```

 Step 4: Fix any missing dependencies (if needed)
```sudo apt-get install -f```

#Step 5: Launch Parsec
 You can start Parsec from your application menu or by typing:
parsec

 Step 6: Sign in or create an account
 You’ll need a Parsec account to use the app. Sign in or register if you don’t have one.

 Step 7: Configure settings (optional)
 Adjust resolution, bandwidth, and controller settings as needed in the app’s settings menu.
